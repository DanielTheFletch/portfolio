My implementation of a basic palindrome checker in the JavaScript language. This was my first hands-on project where I got to really dig into JavaScript programming, especially when it came to exploring ES6+ features. The CodePen link above will open an in-browser code editor and console. Tinkering at the console will enable you to interface with the  palindrome  function I implemented. Try testing out some of my personal favorite palindromes:

  • "A BUTT TUBA"
  • "RATS LIVE ON NO EVIL STAR"
  • "YO, BANANA BOY!"
